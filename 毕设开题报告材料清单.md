## 🎯 快速开始（三步完成）

### 第一步：生成架构图

**方法A：在线生成（最简单）**

1. 访问 https://mermaid.live/
2. 打开 `系统架构图.md`
3. 复制图1-4的代码（```` ```mermaid ... ```` 之间）
4. 粘贴到网页左侧编辑器
5. 点击右上角 "Download PNG"
6. 保存到 `diagrams/` 文件夹

**方法B：本地生成**

```bash
# 安装mermaid-cli
npm install -g @mermaid-js/mermaid-cli

# 运行生成脚本
python generate_diagrams.py
```

### 第二步：转换为Word

**方法A：使用Pandoc（推荐）**

```bash
# 安装Pandoc
# 从 https://pandoc.org/installing.html 下载安装

# 转换
pandoc "开题报告-技术文档智能问答系统-完整版.md" -o "开题报告.docx"
```

**方法B：手动复制**

1. 打开Microsoft Word
2. 复制Markdown文件内容
3. 粘贴到Word
4. 应用样式（见下文格式设置）

### 第三步：插入图片并调整格式

1. 在Word中插入生成的PNG图片
2. 添加图注（图1、图2...）
3. 调整格式（见下文格式要求）

## 📐 格式要求（Word）

### 页面设置

```
纸张: A4 (210 × 297 mm)
页边距:
  上下: 2.54 cm
  左右: 3.17 cm
```

### 字体设置

| 元素 | 字体 | 字号 | 备注 |
|------|------|------|------|
| 正文 | 宋体 | 小四 (12pt) | 1.5倍行距 |
| 标题1 | 黑体 | 三号 (16pt) | 段前段后各0.5行 |
| 标题2 | 黑体 | 四号 (14pt) | 段前段后各0.5行 |
| 标题3 | 黑体 | 小四 (12pt) | 加粗 |
| 图注 | 宋体 | 小五 (10.5pt) | 居中，图下方 |
| 表注 | 宋体 | 小五 (10.5pt) | 居中，表上方 |

### 图片要求

- 格式：PNG（推荐）
- 分辨率：≥150 DPI，推荐300 DPI
- 位置：居中
- 大小：占页面宽度80-100%
- 编号：图1、图2、图3...
- 标注：图名居中，位于图片下方

## 📞 常见问题

### Q1: Mermaid图显示异常？
**A**: 使用在线工具 https://mermaid.live/ 或安装 mermaid-cli

### Q2: 转换Word后格式乱了？
**A**:
1. 使用Pandoc转换（推荐）
2. 或手动应用Word样式
3. 确保原Markdown格式规范

### Q3: 图片分辨率不够？
**A**: 生成时设置更高DPI
```bash
mmdc -i diagram.mmd -o diagram.png -s 3  # 3倍缩放
```

### Q4: 需要调整哪些内容？
**A**: 根据实际情况修改：
